# MacKR

**macOS에서 한글 입력이 깨지는 앱들을 자동으로 보정하는 유틸리티**

---

## 이런 문제가 있다면?

- 한글 입력 시 **"한글"** 이 **"한ㅡ"** 로 나옴
- 입력 중인 글자가 **안 보임**
- 글자마다 **방향키**를 눌러야 제대로 입력됨
- 자음/모음이 풀려서 나옴

이 앱을 설치하면 바로 해결됩니다!

---

## 어떤 앱에서 발생하나요?

macOS의 한글 IME 조합(composition)을 제대로 지원하지 않는 앱에서 발생합니다.

| 앱 | 증상 |
|----|------|
| **CorelDRAW** | 조합 중 글자 안 보임, "한글" → "한ㅡ" |
| **Wine / CrossOver 앱** | 자모가 풀어쓰기로 분리됨 (ㅎㅏㄴ) |
| **IntelliJ IDEA** (JCEF 영역) | 내장 브라우저에서 한글 입력 깨짐 |

> 이 외에도 한글 입력이 깨지는 앱이라면 **직접 등록해서 사용** 가능합니다.

## 설치 방법

### 1단계: 다운로드

[여기서 다운로드](../../releases/latest) → **MacKR_Installer.pkg** 클릭

### 2단계: 설치

1. 다운로드한 **PKG 파일 더블클릭**
2. **"계속"** → **"설치"** 클릭
3. 설치 완료!

> **"확인되지 않은 개발자" 경고가 뜨면:**
>
> `시스템 설정` → `개인정보 보호 및 보안` → 아래쪽에 **"확인 없이 열기"** 클릭

### 3단계: 권한 설정 (필수!)

앱이 키보드 입력을 보정하려면 **손쉬운 사용** 권한이 필요합니다.

1. 메뉴바(화면 맨 위, 시계 옆)에서 **"한"** 클릭
2. **"권한 설정 열기"** 클릭
3. `시스템 설정` → `개인정보 보호 및 보안` → `손쉬운 사용` 이 열림
4. 목록에 **MacKR**가 없으면:
   - 아래 **+ 버튼** 클릭
   - `/Applications/MacKR.app` 선택
5. **MacKR** 토글 **켜기**

### 4단계: 대상 앱 등록

1. 메뉴바 **"한"** 클릭
2. **"+ 앱 추가..."** 클릭 → 한글 깨지는 앱 선택
3. 또는 해당 앱을 연 상태에서 **"+ 현재 앱 추가"** 클릭

등록한 앱이 활성화되면 자동으로 한글 보정이 작동합니다.
등록하지 않은 앱에서는 전혀 개입하지 않습니다.

## 삭제 방법

### 방법 1: 앱 내에서 삭제

메뉴바 **"한"** 클릭 → **"앱 삭제 (Uninstall)"**

### 방법 2: 더블클릭 삭제

`uninstall.command` 파일을 더블클릭 → 자동 삭제

### 방법 3: 수동 삭제

```
killall MacKR
```
```
sudo rm -rf /Applications/MacKR.app
```
```
sudo pkgutil --forget com.mackr.app
```

## 동작 원리

일부 macOS 앱은 한글 IME의 조합(composition) 상태를 제대로 처리하지 못합니다.
macOS의 `NSTextInputClient` 프로토콜을 올바르게 구현하지 않은 앱에서 발생하는 공통적인 문제입니다.

MacKR은:
1. 등록된 대상 앱이 활성화되면 키보드 입력을 가로채고
2. 자체 한글 조합 엔진으로 자모를 조합한 뒤
3. 완성된 한글 문자를 앱에 직접 전달합니다

등록하지 않은 앱에서는 전혀 개입하지 않습니다.
